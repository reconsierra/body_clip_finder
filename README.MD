
# Body Clip Finder

A Streamlit web app to help sales reps and customers find matching body clips easily and enjoyably.

## Features

- Dropdown filters for:
  - Colour
  - OEM Brand
  - Suit Hole Size
  - Body Clip Type
- Card-style layout displaying:
  - Product image
  - Description
  - Dimensions
  - OEM references

## Styling

- Font: Calibri
- Color Palette:
  - Background: `#FFFFFF`
  - Accent: `#CC0000`
  - Cards: `#BFBFBF`
  - Text: `#737373` and `#000000`

## Mobile Optimization

- Designed to open cleanly on iPhone 14 screens
- Responsive layout with minimal padding and spacing

## Favicon Setup

- Favicon included using WÜRTH logo
- Automatically applied via `st.set_page_config(page_icon='favicon.png')`
